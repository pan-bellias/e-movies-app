Welcome to the e-movies!

If you already have an account, go ahead and log in. If you are new to e-movies, get started by creating a viewer account or a manager account.

What's this about?

This Django application is for the semester exercise in the context of 'Basic DevOps Concepts and Tools' course at DIT HUA. In this application, users can sign up as a viewer or a manager. Managers can create and delete movies and viewers can view and rate movies.

Want to run this code locally? Read detailed instructions on how to run this project.
